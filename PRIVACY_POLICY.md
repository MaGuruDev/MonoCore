## Privacy Policy
### MaGuru — MaGuru Monobank Core Module for Magento 2
Last updated: June 2026

This Privacy Policy describes how the MaGuru Monobank Core Module for Magento 2 extension handles data.

### External Service: Monobank
This extension integrates with **Monobank** (api.monobank.ua), a Ukrainian
banking and payment service. As part of this integration, the following data
may be transmitted to Monobank servers:

- Order amount, currency, and reference number
- Customer phone number (for Monobank Частинами / BNPL)
- Payment status and transaction identifiers
- Webhook URLs for payment status callbacks
- Fiscal receipt data (for MonoFiscal component)

Data is transmitted only when the integration is enabled and configured with
a valid Monobank merchant API token. Monobank's processing of personal data
is governed by Monobank's Privacy Policy and applicable Ukrainian law.

### No Data Collection by MaGuru
MaGuru does not collect, store, or receive any payment or personal data.
All data flows directly between your Magento installation and Monobank.
